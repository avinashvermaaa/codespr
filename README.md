# CodeSphere :- An Online AI Based Compiler & Coding Platform.
* https://codespr.netlify.app/

# Install dependencies 
```
npm install
```
# Backend
* cd backend
```  
    node server.js
    {It should run the server.js in port you defined in .env}
```
# Frontend

* cd frontend
```
    npm start     
    {It should run in port : 3000}
```
